# BlogWebsite-Django
A simple blog website built with Django that allows users to view blog posts, categories, and comments. The project includes basic database models, Django admin integration, and multiple HTML templates for displaying users, posts, comments, and categories.
- Built with Django
- Manages users, blog posts, comments, and categories
- Uses Django ORM for database relationships
- Includes admin panel for content management
- Basic HTML templates for frontend rendering

Tech Stack:

- Python
- Django
- SQLite
- HTML

Features:

- View list of blog posts
- View post details
- Display comments linked to posts
- Organize posts by categories
- Admin dashboard for data management
